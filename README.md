# Sanskar Yadav Portfolio (Fixed)

This React app references images from `src/assets` instead of `public`.

## Run

1. Install dependencies:
   ```
   npm install
   ```

2. Start:
   ```
   npm start
   ```

## Structure

- `public/`
  - `index.html`
- `src/`
  - `assets/` (background images)
  - `components/`
  - `styles/`
  - `App.js`, `App.css`
  - `index.js`, `index.css`
